1. Get the TOTP QR code.
2. Install zbar, decode the QR code.

```bash
brew install zbar
zbarimg qr.png
```

3. Generate totp code from the decoded code.

```bash
from dpdispatcher.utils import generate_totp
print(generate_totp("A5UKDJDSDSK"))
```
